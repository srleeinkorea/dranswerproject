환자 모니터링 및 챗봇 시스템

Monitoring, Not Prediction

벤틸레이션 신호 분석과 PRO 데이터를 기반으로 환자의 현재 상태를 정량화하고, 변화 패턴을 분석하는 통합 모니터링 플랫폼입니다.

시스템 개요

본 시스템은 장기 기계호흡 환자의 안정적 관리를 위해 설계되었습니다. 홈 벤틸레이터로부터 수집한 호흡 신호와 보호자가 제공하는 임상 정보를 통합 분석하여 환자의 상태 변화를 감지하고 적절한 임상 개입을 지원합니다.

핵심 데이터 수집 주기 및 흐름

Ventilation Signal – 1시간 주기 Hourly
PRO Data – 보호자 주간 입력 Weekly
Event Trigger – 긴급 상황 판단용 Realtime

벤틸레이션 신호 분석

데이터 수집 프로토콜

호흡 신호는 1시간 단위로 정기적으로 수집되어 안정화된 상태의 환자 데이터를 확보합니다. 이러한 간격은 일시적 변동을 최소화하고 임상적으로 의미 있는 패턴 변화를 감지하는 데 최적화하도록 설계합니다.

데이터 처리 방법론

수집된 신호 데이터는 다단계 처리 프로세스를 거칩니다. 먼저 기기 노이즈나 순간적인 이상치를 제거하고, 환자의 기준값 대비 상대적 변화도를 계산합니다. 이후 지난 7일 데이터를 기반으로 추세 분석을 수행하여 점진적인 악화 신호를 감지합니다.

PRO 데이터 – 보호자 임상 관찰

주간 건강 평가

보호자는 주 1회 체계적인 형식으로 환자의 임상 상태를 보고합니다. 이러한 정성적 정보는 벤틸레이션 신호의 정량적 데이터를 보완하여 포괄적인 임상 평가를 가능하게 합니다.

평가 항목

  • 호흡기 상태: 산소포화도, 호흡곤란, 분비물 변화, 기침 빈도
  • 감염 징후: 체온, 분비물 색상 및 성상
  • 일상 활동: 수면 질, 식욕, 활동 수준
  • 의료 개입: 약물 변경, 시술, 병원 방문
  • 환자 개별 진단명에 따른 상태: 발작, 경련, 의식 수준 변화

긴급 상황 에스컬레이션 체계

일상적 주간 보고 외에도 보호자는 긴급 상황을 즉시 시스템에 보고 - 의료진 즉각 개입이 필요한 신호는 즉각 red signal 제시

  • 급성 호흡곤란 (청색증)
  • 고열 (38°C 이상)
  • 의식 변화 또는 반응성 저하

신호 모니터링/분석 & 위험도 분류 엔진(머신러닝)

시스템 아키텍처

위험도 분석 엔진은 벤틸레이션 신호와 PRO 데이터를 통합하여 환자의 현재 상태를 평가합니다. Intent 분류기가 사용자 입력을 먼저 분류한 후, Orchestration 모듈이 관련 데이터를 수집하고 종합합니다.

트리아지 시스템:Traffic light triage system

레벨 임상적 의미 특성 대응
🟢 Green 정상 범위 신호 안정적, 편차 최소 일상적 모니터링
🟠 Yellow 경계 상태 점진적 변화, 추세 악화 강화된 관찰
🔴 Red 위험 상태 급격한 변화, 임상적 우려 의료진 즉시 개입

LLM 엔진 적용 인터페이스

챗봇 기반 안내

LLM 기반 챗봇은 위험도 분석 엔진의 결과를 환자 친화적인 자연어로 변환합니다. 의료 판단을 제공하지 않으면서도 상태 변화를 이해하기 쉬운 방식으로 설명하고, 필요한 임상 조치를 안내합니다.

✓ 안전성 기능: 의료 판단 표현 자동 차단, 응급 상황 조건부 알림, 의료진 연계 권고 문구 필수 포함

응답 예시

입력: "요즘 호흡 상태가 어떤가요? 분석: - 의도 분류: 상태 문의 - 벤틸레이션: RR ↑, MinVent ↓ - PRO: 호흡곤란 증상 보고 - 위험도: Yellow(경계) LLM 응답: "최근 호흡 패턴에 변화가 감지되었습니다. 의료진에게 알리고 현재 상태를 계속 관찰하겠습니다. 긴급 상황 발생 시 바로 알려주세요."

시간별 모니터링 주기

매시간 벤틸레이션 신호가 자동으로 수집되어 위험도 평가가 이루어집니다.

주간 평가 일정

매주 정해진 시간에 보호자는 건강 상태 평가 양식을 작성합니다. 누적된 벤틸레이션 데이터와 통합 분석되어 더욱 정교한 임상 평가가 가능합니다.

긴급 상황 대응

긴급 상황 발생 시 보호자의 즉시 보고는 현재 벤틸레이션 신호 및 PRO 데이터와 함께 종합 분석됩니다. Red 등급 시 응급실 권고 프로토콜이 자동 실행됩니다.
