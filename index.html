환자 모니터링 및 챗봇 시스템

Patient Monitoring & Chatbot System

벤틸레이션 신호 분석과 PRO 데이터 기반의 통합 환자 모니터링 플랫폼

시스템 개요

플랫폼 아키텍처

본 시스템은 장기 기계호흡 환자의 안정적 관리를 위해 설계되었습니다. 홈 벤틸레이터로부터 수집한 호흡 신호와 보호자가 제공하는 임상 정보를 통합적으로 분석하여, 환자의 상태 변화를 조기에 감지하고 적절한 임상 개입을 지원합니다.

벤틸레이션 신호 분석

데이터 수집 프로토콜

호흡 신호는 1시간 단위로 정기적으로 수집되어 안정화된 상태의 환자 데이터를 확보합니다. 이러한 간격은 일시적 변동을 최소화하고 임상적으로 의미 있는 패턴 변화를 감지하는 데 최적화되어 있습니다.

데이터 처리 방법론

수집된 신호 데이터는 다단계 처리 프로세스를 거칩니다. 먼저 기기 노이즈나 순간적인 이상치를 제거하고, 환자의 기준값 대비 상대적 변화도를 계산합니다. 이후 지난 7일 데이터를 기반으로 추세 분석을 수행하여 점진적인 악화 신호를 감지합니다.

PRO 데이터 – 보호자 임상 관찰

주간 건강 평가

보호자는 주 1회 체계적인 형식으로 환자의 임상 상태를 보고합니다. 이러한 정성적 정보는 벤틸레이션 신호의 정량적 데이터를 보완하여 포괄적인 임상 평가를 가능하게 합니다.

평가 항목

  • 호흡기 상태: 호흡곤란, 분비물 변화, 기침 빈도
  • 감염 징후: 체온, 분비물 색상 및 성상
  • 일상 활동: 수면 질, 식욕, 활동 수준
  • 의료 개입: 약물 변경, 시술, 병원 방문
  • 환자 진단명에 따른 추가 입력 항목 고려 검토
    (예) 신경학적 상태: 발작, 경련, 의식 수준 변화

긴급 상황 즉시 보고

일상적 주간 보고 외에도 보호자는 긴급 상황을 즉시 시스템에 보고할 수 있습니다. 이러한 즉시 보고는 실시간 의료진 개입을 촉발할 수 있는 중요한 임상 신호입니다.

  • 급성 호흡곤란(청색증)
  • 고열 (38°C 이상)
  • 의식 변화 또는 반응성 저하

위험도 분석 엔진

시스템 아키텍처

위험도 분석 엔진은 벤틸레이션 신호와 PRO 데이터를 통합하여 환자의 현재 상태를 평가합니다. Intent 분류기가 사용자 입력을 먼저 분류한 후, Orchestration 모듈이 관련 데이터를 수집하고 종합합니다.

처리 파이프라인

1
사용자 입력 분류
입력을 상태 문의, 이벤트 신고, 일반 질문으로 분류합니다.
2
데이터 통합
최신 벤틸레이션 신호와 PRO 데이터를 수집 및 통합합니다.
3
위험도 예측
머신러닝 모델을 통한 위험도 점수를 산출합니다.
4
트리아지 분류
위험도 점수를 신호등 레벨(Green/Yellow/Red)로 변환합니다.

신호등 트리아지 시스템

레벨 임상적 의미 특성 대응
🟢 Green 정상 범위 신호 안정적, 편차 최소 일상적 모니터링
🟠 Yellow 경계 상태 점진적 변화, 추세 악화 강화된 관찰
🔴 Red 위험 상태 급격한 변화, 임상적 우려 의료진 즉시 개입

챗봇 통합

LLM 기반 챗봇은 위험도 분석 엔진의 결과를 환자 친화적인 자연어로 변환합니다. 의료 판단을 제공하지 않으면서도 상태 변화를 이해하기 쉬운 방식으로 설명하고, 필요한 임상 조치를 안내합니다.

안전성 메커니즘

모든 응답은 자동 검증 프로세스를 거칩니다. 의료 진단이나 처방으로 해석될 수 있는 표현은 제거되며, 환자의 안전을 위해 의료진의 최종 판단을 항상 강조합니다.

✓ 안전성 기능: 의료 판단 표현 자동 차단, 응급 상황 조건부 알림, 의료진 연계 권고 문구 필수 포함

응답 예시

입력: "요즘 호흡이 좀 어렵다고 해요" 분석: - 의도 분류: 상태 문의 - 벤틸레이션: RR ↑, MinVent ↓ - PRO: 호흡곤란 증상 보고 - 위험도: Yellow (경계) 응답: "최근 호흡 패턴에 변화가 감지되었습니다. 의료진에게 알리고 현재 상태를 계속 관찰하겠습니다. 긴급 상황 발생 시 바로 알려주세요."

일일 운영 워크플로우

시간별 모니터링 주기

매시간 벤틸레이션 신호가 자동으로 수집되어 실시간 위험도 평가가 이루어집니다. Yellow 또는 Red 신호가 감지될 경우 보호자와 의료진에게 즉시 알림이 발송됩니다.

주간 평가 일정

매주 정해진 시간에 보호자는 건강 상태 평가 양식을 작성합니다. 이러한 정보와 누적된 벤틸레이션 데이터가 통합되어 더욱 정교한 임상 평가가 가능해집니다.

긴급 상황 대응

긴급 상황 발생 시 보호자의 즉시 보고는 현재 벤틸레이션 신호 및 PRO 데이터와 함께 종합 분석됩니다. Red 판정 시 의료진 호출 프로토콜이 자동으로 실행됩니다.
