# dranswerproject

📋 핵심 설명 (요약)
왜 일일이 아닌 주간인가?
1️⃣ 이중 모니터링 체계 (Dual Monitoring System)
Ventilation Signal (1시간)  → 객관적, 자동화, 실시간
                              급성 변화 감지 (회로 폐쇄, RR↑ 등)

PRO Data (주간)             → 주관적, 수동, 정기적
                              만성 추세 파악 (발작, 감염 등)
2️⃣ 5가지 핵심 이유
이유설명보호자 부담일일 입력은 순응도 저하 → 데이터 품질 악화데이터 품질주간 체계적 평가 > 일일 산발적 입력벤틸의 보완1시간 신호로 이미 고주파 모니터링 가능임상 의미점진적 변화는 주간 단위로 충분히 감지시스템 효율365배의 데이터 증가 → 거짓 경보 증가
3️⃣ 실제 워크플로우
1시간마다    → 벤틸 신호 수집 → Yellow/Red 즉시 알림
1일 1회      → 임상가 신호 트렌드 검토
주 1회      → PRO 입력 + 7일 누적 데이터 종합 분석
긴급 발생   → 즉시 이벤트 채널로 처리 (주간 입력 무관)
4️⃣ 주요 차이점
일일 PRO:  부담 큼 → 순응도 낮음 → 데이터 질 낮음 → 노이즈 증가
주간 PRO:  부담 적음 → 순응도 높음 → 데이터 질 높음 → 신호 명확
