통계

LLM 기반 임상 설명 인터페이스

LLM 기반 챗봇은 위험도 분석 결과를 환자 친화적인 자연어로 변환합니다. 의료 판단 표현은 자동 차단되며, 의료진의 최종 판단을 항상 강조합니다.

입력: "요즘 호흡이 좀 어렵다고 해요" 분석: - 의도 분류: 상태 문의 - Ventilation: RR 증가, MinVent 감소 - PRO: 호흡곤란 보고 - 위험도: Yellow (경계) 응답: "최근 호흡 패턴 변화가 감지되었습니다. 의료진에게 알리고 상태를 지속적으로 관찰하겠습니다. 긴급 상황 발생 시 즉시 알려주세요."
